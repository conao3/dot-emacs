# .emacs.d

This repository is my private settings for Emacs.

## Filename rules

[inits/*](https://github.com/handlename/dot-emacs/tree/master/inits) filename prefix:

* 00: fundamental settings
* 10: pre load settings
* 20: custom functions
* 30: utils
* 40: minor mode
* 50: major mode
* 90: post load settings
